Welcome to the final marking program for the C Project for 2022 - "Boulder Break".  There are 10 tasks, and each task is worth 10 marks.  Pay attention to the information on this page which explains how the project is marked.  

Please note:

  • "Precheck" is enabled.  This allows you to check your submissions for any syntax errors or syntax-related warnings before making a submission.  This is designed to help you avoid very minor syntax issues that can be frustrating.  There is no penalty for making "Prechecks". Once your code passes the "Precheck", you can have confidence that it will execute against the test cases.
  • Penalties: this is the final project for the course.  You should be thoroughly testing your code before you submit it.  The penalty scheme is different to what you are used to in the lab.  The penalty scheme is: 5, 10, 20, 30, 40, 50, 60, 70, 80.  In other words, your first incorrect submission (that fails any tests) attracts a 5% penalty.  Subsequent submissions attract penalties in multiples of 10 up to a maximum of 80%.  In order to earn the marks for a task, all test cases must pass.
  • Tasks marked one at a time.  For each task, you will be asked to submit a subset of the functions you have defined - the functions to submit will be clearly labelled for each task.  You may submit helper functions that you have written, in addition to the required functions.  If you do submit helper functions, pay attention to the order in which the functions are defined.  The C compiler works from the top down, so you must define functions before you call them.  In other words, you should provide the definition to your helper function(s) before the definition of any functions that call the helper function(s).
  • Hidden tests.  Some of the tests for each task will be visible, and some will be hidden.  The hidden tests are not published.
  • Submit all and finish.  When you have finished marking your project, you will see the "Finish attempt" option at the bottom of the page for Task Ten.  Please make sure you click "Finish attempt", and then the "Submit all and finish" button that appears before the project marking deadline.  After you have clicked "Submit all and finish" you will be shown a summary of your marks for the project.

      

Good luck!
